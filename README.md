# P-LearnWitchRunes
Personal Test Project as an introduction to Java Swing and learn to read the Witch Runes from Madoka Magica. :D

Hello, this isn't a professional project. It's testing myself and my ability to learn new topics I find important
while doing something I am interested in. In this first version, I am trying to learn the GUI and structure of 
Java Swing on Eclipse. My goal was to create a small game to help with my memorization of the Witch Runes.

Current Flaws: 
- Transfom screens allows English Alphabet Only. No numbers or spaces.
- Up to 10 characters only
